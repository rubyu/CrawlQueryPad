
CQPad は、WEB上の任意の領域を切り出し、柔軟に
加工するためのツールです。


使い方：
まず、クローラを記述します。
とりあえず、次の一文を入力してみましょう。

"http://www.google.co.jp/"

記述に問題がなければ、Crawlボタンがアクティブになります。
Crawlボタンを押します。

クローラがURLを集め終わると、その結果を用い、出力を描写するプラグインが呼ばれます。
描写が終わると、右下のSubmitボタンがアクティブになります。
Submitボタンを押します。描写の結果を保存するプラグインが呼ばれます。

結果を任意の場所に保存します。

大まかな流れは以上です。
クローラの記述にさえ慣れてしまえば、各プラグインは自由にカスタマイズできるため、非常に柔軟です。
